Features That Differentiate Mean

🔁

Leitner Spaced Repetition

Active recall sessions schedule words across 5 progressive confidence boxes. Mastered words are enqueued less frequently, creating long term retention.

🪜

Word Intensity Ladders

Quickly view synonyms ordered by semantic strength. Ascend from common descriptions to complex academic equivalents.

🛡️

Professional Domain Modes

Study specific glossaries for specialized fields. Includes built-in decks for Cybersecurity, Finance, Legal, Medical, and GRE/IELTS.

🎮

Gamified Vocabulary Suite

Learn while playing. Features Hangman, Definition Matchers, Spelling Quizzes, and Flashcards playable in both standard CLI and full-screen TUI.

📡

Local JSON Server

Spin up a local REST API endpoint on localhost:8080 with mean serve. Programmatic queries get immediate definition formats.

🐚

Unix Pipe Support

Supports stdin pipe flows. Stream outputs directly from other utilities (e.g. echo "recursion" | mean --json).
